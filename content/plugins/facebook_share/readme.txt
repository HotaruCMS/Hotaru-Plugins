Facebook Share Plugin for Hotaru CMS
--------------------------------
Created by: William Dahlheim
Extended by: Kyle Carlson

Description
-----------This plugin adds a Facebook Share button in the extra field of each post.

Instructions
------------
1. Upload the "facebook_share" folder to your plugins folder. Install it as usual from Plugin Management in Admin.
