Description
-----------
My first attempt at a plugin.  It's a lightweight module to display basic stats about a Hotaru site.

Suggestions & feedback are always appreciated!


Instructions
------------
1. Upload the "stats" folder to your plugins folder. 
2. Install it from Plugin Management in Admin.
3. Activate "Stats Display" in the Sidebar Widgets plugin, and order it how you chose
